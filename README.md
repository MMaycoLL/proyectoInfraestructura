# Proyecto Final Infraestructura Computacional

script de instalacion y visualizacion de las siguientes opciones

- Instalar e iniciar un servidor web Nginx
- Instalar e iniciar un servidor web Http
- Consultar el release del sistema
- Consultar el servidor Dns del servidor
- Consultar el gateway del servidor
- Consultar el estado de setLinux
- Listar las opciones del firewall
- Mostar la informacion del archivo /etc/passwd
- Imprimir un mensaje ingresado
- Listar los discos duros del sistema
- Saber el tiempo que lleva prendido el servidor
- Listar los procesos del servidor
- Modificar el index Nginx
- Modificar el index Http

**Nota:** para la visualizacion de los index de los servidores debera ingresar a la siguiente url _http://localhost/_
